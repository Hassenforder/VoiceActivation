History

December 2020 : Initial idea
A lost article from Medium about voice recognition ready to use in browser
How to Use Speech Recognition and Speech Synthesis in JavaScript
Ravindu Senaratne
Initial draft of the voice activation requirements
January - June 2021 : Start of the proof of concept
Two students work on the POC
  • Amal Matmati
  • Maxence Azad Markarian
Second draft of the voice activation requirements
At the end of the project it works well in French
July 2021 : Rewriting of the API
Third draft of the voice activation requirements
Based on the POC, I rewrite the API to have better features :
  • Basic (in french) synonyms, derivations, ...
  • More basic actions such as select, check, uncheck, style, etc...
  • rely only on javascript
  • remove jquery, leave bootstrap
  • add basic UI and debug on a separate library
API version 1 is out of the box
September 2021 : Extension of the API
A way to the next level :
  • better API about langages (synonyms, etc)
  • more basic actions
  • more generic activation rules
  • activation state machine
