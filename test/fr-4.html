Exemple de formulaire Activable à la voix

Point de vue développeur

L'objectif de cette page est de montrer l'activation de plusieurs formulaires comportant des champs communs. Mais, les formulaires peuvent être cachés/dévoilés selon une logique métier (ici on click sur un bouton). Il y a un formulaire à droite et un formulaire à gauche.
Les règles d'activations ciblent spécifiquement les formulaires. Il y a donc trois fois les même règles mais pas pour les mêmes cibles. Quand on prononce les mots ce ne seront que les formulaires visiblent qui seront renseignés.

Point de vue utilisateur

Dans cette page, il faut parler Français. Les verbes sont à l'infinitif ou à la deuxième personne. Le point rouge dans le titre montre la reconaissance de mots. Après un silence il se relance automatiquement, il faut arrêter manuellement la reconnaissance. Le champs mote de passe est un simple text pour facilement visualiser le contenu.

Sur le coté droit de la page, un formulaire de connexion, il est possible de prononcer :
  • remplir nom <<nom(s)>>
  • remplir password <<quelque chose>>
    -- lisible car c'est un text et pas un password
  • changer se souvenir
    -- bascule la case à cocher
  • connecter moi
  • changer ballon
    -- permet d'afficher/cacher les ballons
  • changer debug
    -- permet d'afficher/cacher la fenêtre de debug
  • changer voix
    -- permet de cacher le bouton stop mais la commande vocale est toujours valide
  • stop voix
