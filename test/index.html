Samples

Several samples are proposed for each language
  1. A very simple form which relevant elements are activated
  2. Same form but with a global activation in one special element
  3. Same form without previous activations but with a UI (balloons & debugger) and some new activations about color and ink to build interactively the ugliest form
  4. Three times the same form but hidden on sides for popup with same activations
  5. A grid embedded in the page, but the grid is generated by a foreign tool, so no access to the button element nor no id
  6. A tabs with a grid in each of the tab, activated using different rules using the same vocabulary
  7. Same tabs as previous but with a 'generic' rule set
  8. Two grids embedded in the page, a way to differentiate target using special subjects
  9. Two grids embedded in the page with a state machine (first sentence to select, several sentences to act, a last to unselect)
