Developper perspective

L'objectif de cette page est de montrer quand adaptant le vocabulaire il est possible d'activer plusieurs grilles toujours visibles dans la même page.
Euh pas cool, pour l'instant ... les deux grilles réagissent en même temps.

User perspective

In this page, you have to talk in english, colour have to known by the browser. Infinitive verbs or second person. The red dot in the title means recognizer is processing, after a silence it stops automatically.

On the right of this page a tab with two panes with an activated grid :
  • fill title <<new title>>
  • fill search <<letter(s)>>
    -- can't add automatically the <enter>
  • sort 1 or sort id
    -- use id col to sort the grid and/or toggle order
  • sort 2 ou sort sender
    -- use sender col to sort the grid and/or toggle order
  • sort 3 ou sort received
    -- use received col to sort the grid and/or toggle order
  • go begin
    -- goto first page
  • go before
    -- goto previous page
  • go(page) 1
    -- goto page 1
  • go (page) 2
    -- goto 2
  • go (page) 3
    -- goto 3
  • go (page) 4
    -- goto 4
  • go (page) 5
    -- goto 5
  • go next
    -- goto next page
  • go last (page)
    -- goto last page
  • edit 1 or edit first
    -- opens edit popup for the first row
  • edit 2 or edit second
    -- opens edit popup for the second row
  • edit 3 or edit third
    -- opens edit popup for the third row
  • edit 4 or edit fourth
    -- opens edit popup for the fourth row
  • edit 5 or edit fifth
    -- opens edit popup for the fifth row
  • delete 1
    -- opens delete popup for the first row
  • delete 2
    -- opens delete popup for the second row
  • delete 3
    -- opens delete popup for the third row
  • delete 4
    -- opens delete popup for the fourth row
  • delete 5
    -- opens delete popup for the fifth row
  • stop voice
