Par défaut la page n'est pas activée

Point de vue développeur

L'objectif de cette page est de montrer l'activation en plaçant les règles dans un élément spécial <vai> juste après le <body>.
Un deuxième plugin est utilisé pour afficher :
  • Dans un ballon les phrase que l'on peut prononcer. Pour réduire la taille du ballon, seul le premier mot est affiché un .. montre qu'il y a des synonymes.
  • Dans une fenêtre de 'debug' l'avancement du processs d'identification. Elle permet de voir les différentes étapes et de comprendre ce qui n'à pas marché correctement. Par exemple : 'encrer entrée red' est clairement imprononcable

Point de vue utilisateur

Dans cette page, il faut parler Français sauf pour le nom de la couleur qui doit être en Anglais et qui doit être une couleur connue par le navigateur. Les verbes sont à l'infinitif ou à la deuxième personne. Le point rouge dans le titre montre la reconaissance de mots. Après un silence il se relance automatiquement, il faut arrêter manuellement la reconnaissance.

Sur le coté droit de la page, un formulaire de connexion, il est possible de prononcer :
  • colorier page <<nom couleur >>
  • colorier titre <<nom couleur >>
  • colorier champs <<nom couleur >>
  • colorier entrées <<nom couleur >>
  • encrer entrées <<nom couleur >>
  • fournisseur normal
    -- le selecteur utilise l'option normal
  • fournisseur google
    -- le selecteur utilise l'option google
  • fournisseur facebook
    -- le selecteur utilise l'option facebook
  • fournisseur SSO
    -- le selecteur utilise l'option SSO
  • changer ballon
    -- permet d'afficher/cacher les ballons
  • changer debug
    -- permet d'afficher/cacher la fenêtre de debug
  • changer voix
    -- permet de cacher le bouton stop mais la commande vocale est toujours valide
