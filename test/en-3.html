About

English form sample of Voice Activated

By default the page is not activated

Developper perspective

The purpose of this sample is to activate inputs putting all rules just after the <body> in a special <vai> element. A second plugin is used to display :
  • In a balloon the sentence we can pronounce. To reduce the size of the balloon, only the first word is displayed and an ellipsis shows that synonyms are defined.
  • In a debug window, identification process is shown. It allow to understand the different steps are what was wrong. May be ink entry red which is hard to pronounce.

User perspective

In this page, you have to talk in english, colour have to known by the browser. Infinitive verbs or second person. The red dot in the title means recognizer is processing, after a silence it restarts automatically. It has to be stopped manually.

On the right of this page it looks like a register page, you can say :
  • paint page <<colour name>>
  • paint title <<colour name >>
  • paint field <<colour name >>
  • paint entry <<colour name >>
  • ink entry <<colour name >>
  • provider normal
