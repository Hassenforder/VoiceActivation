Exemple de formulaire Activable à la voix

Par défaut la page n'est pas activée

Point de vue développeur

L'objectif de cette page est de montrer l'activation en plaçant les règles à coté ou dans un élément.

Point de vue utilisateur

Dans cette page, il faut parler Français. Les verbes sont à l'infinitif ou à la deuxième personne. Le point rouge dans le titre montre la reconaissance de mots, après un silence il stoppe automatiquement.

Sur le coté droit de la page, un formulaire de connexion, il est possible de prononcer :
  • remplir titre <<nouveau titre>>
  • remplir nom <<nom(s)>>
  • remplir un <<nom(s)>>
    -- si alt 1 est visible et autorisé le champs nom un est rempli
  • remplir deux <<nom(s)>>
    -- si alt 2 est visible et autorisé le champs nom deux est rempli
  • remplir alternatif <<nom(s)>>
    -- si les deux champs sont visibles et autorisés ils seront remplis ensemble
  • remplir autre <<nom(s)>>
    -- si les deux champs sont visibles et autorisés ils seront remplis ensemble
  • remplir password <<quelque chose>>
    -- pas lisible mais probablement rempli de travers
  • cocher cacher
    -- coche uniquement la case à cocher
  • décocher cacher
    -- décoche uniquement la case à cocher
  • activer désactiver
    -- coche uniquement la case à cocher
  • désactiver désactiver
    -- décoche uniquement la case à cocher
  • changer se souvenir
    -- bascule la case à cocher
  • connecter moi
  • stop voix
