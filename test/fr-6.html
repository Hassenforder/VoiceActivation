Exemple de formulaire Activable à la voix

Par défaut la page n'est pas activée

Point de vue développeur

L'objectif de cette page est de montrer qu'un affichage a onglet est parfaitement pris en compte. Il y a autant de règles pour la grille dans l'onglet 1 que pour la grille de l'onglet 2.
C'est comme si les règles avaient été générées indépendemment pour chaque grille. C'est la page visible qui est activée même si le vocabulaire est le même pour les deux grilles.

Point de vue utilisateur

Dans cette page, il faut parler Français. Les verbes sont à l'infinitif ou à la deuxième personne. Le point rouge dans le titre montre la reconaissance de mots, après un silence il stoppe automatiquement.

Sur le coté droit de la page, un tableau activé :
il y a des synonymes cachés pour tromper le recogniseur et faciliter l'utilisateur.
  • remplir titre <<nouveau titre>>
  • remplir recherche <<lettre(s)>>
    -- pas le <enter> automatiquement
  • trier un ou trier id
    -- utilise la première colonne pour trier et change l'ordre
  • trier deux
    -- utilise la deuxième colonne pour trier et change l'ordre
  • trier trois id
    -- utilise la troisième colonne pour trier et change l'ordre
  • aller (au) début
    -- va sur la première page
  • aller avant
    -- va sur la page précédente
  • aller (page) 1
    -- va sur la page 1
  • aller (page) 2
    -- va sur la page 2
  • aller (page) 3
    -- va sur la page 3
  • aller (page) 4
    -- va sur la page 4
  • aller (page) 5
    -- va sur la page 5
  • aller après
    -- va sur la page suivante
  • aller dernière (page)
    -- va sur la dernière page
  • éditer 1
    -- lance le popup qui ferait l'édition de la première ligne
  • éditer 2
    -- lance le popup qui ferait l'édition de la deuixème ligne
  • éditer 3
    -- lance le popup qui ferait l'édition de la troisième ligne

  • éditer 5

  • supprimer 1
