About

English form sample of Voice Activated

By default the page is not activated

Developper perspective

The purpose of this sample is to activate inputs putting the rule near or inside the elements.

User perspective

In this page, you have to talk in english. Infinitive verbs or second person. The red dot in the title means recognizer is processing, after a silence it stops automatically.

On the right of this page it looks like a register page, you can say :
  • fill title <<something>>
  • fill name <<name(s)>>
  • fill one <<name(s)>>
    -- if alternate 1 is visible and enabled the input field is filed
  • fill second <<name(s)>>
    -- if alternate 2 is visible and enabled the input field is filed
  • fill alternate <<name(s)>>
    -- if both are visible and enabled the two fields are filed
  • fill other <<name(s)>>
    -- if both are visible and enabled the two fields are filed
  • fill password <<something>>
    -- not readable ... but done wrongly
  • check hidden
    -- check only
  • uncheck hidden
    -- uncheck only
  • activate deactivate
    -- check only
  • deactivate deactivate
    -- uncheck only
  • change remember
    -- toggle
  • connect me
  • stop voice
